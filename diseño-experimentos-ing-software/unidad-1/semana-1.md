Sistema SIRE: Es un sistema en donde se declara ventas y compras.

### Devops

Es una metodología que busca unir el mundo del desarrollo y de las operaciones.
Con esto se acaba la brecha entre ambos mundos, y se busca que el software se entregue de manera continua, con calidad y con rapidez.

#### Integración continua: 

Es el proceso de integrar el código de manera continua, es decir, cada vez que se hace un cambio en el código, se integra de inmediato.

#### Entrega continua: 

Es el proceso de entregar el software de manera continua, es decir, cada vez que se hace un cambio en el código, se entrega de inmediato.

#### Despliegue continuo: 

Es el proceso de desplegar el software de manera continua, es decir, cada vez que se hace un cambio en el código, se despliega de inmediato.

### DevSecOps

Es una metodología que busca unir el mundo del desarrollo, de las operaciones y de la seguridad. Con esto se busca que el software se entregue de manera continua, con calidad, con rapidez y con seguridad.

Servidor de inteligencia de negocio (BI): Es un servidor que se utiliza para almacenar y analizar datos de una empresa, con el fin de tomar decisiones estratégicas.

#### Terminología

Liquidación de empresa: Es cuando una empresa se encuentra en quiebra y se vende todo lo que tiene para pagar a los acreedores.

Servidor de Active Directory: Es un servicio de directorio que se utiliza para administrar los recursos de una red, como usuarios, grupos, computadoras, etc.

rasomware: Es un software que se utiliza para analizar el código fuente de un programa y detectar posibles errores o vulnerabilidades.

Bussiness Intelligence: Es un conjunto de técnicas y herramientas que se utilizan para analizar los datos de una empresa y tomar decisiones basadas en esos datos.
