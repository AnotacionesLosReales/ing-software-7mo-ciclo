# Semana 3: Tasa de interés compuesto o nominal

## Sesión 1: Interés compuesto o nominal

### Definición 

- Es el interés simple, aplicado de manera sucesiva a un capital que crece a medida que se suman los intereses al capital invertido.

### Periodo de capitalización

- Es el periodo de tiempo fijo donde los intereses ganados se convierten en nuevo capital para el siguiente periodo de tiempo.
- Este periodo de tiempo se constituye en el periodo de acumulación o capitalización.
- En cualquier problema, siempre manda el periodo de capitalización.
- Si no se menciona, se debe asumir que la capitalización se produce de forma **DIARIA**.

### Tasa nominal y efectiva

- Cuando la tasa de interés compuesta se capitaliza más de una vez por año, recibe el nombre de tasa nominal.
- A su respectivo incremento porcentual efectivo respecto de capital inicial recibe el nombre de Tasa Efectiva.

```
X% = ((S / C) - 1) × 100

X: tasa de interés del periodo
```

### Normas a seguir

- La tasa de interés siempre ingresa a las formulas expressda en tanto por ciento (dividida entre 100).
- Cuando no se indica nada de la tasa de interés nomina, se debe asumir que está expresada en años.
- Si la capitalización no se define, se debe asumir que la tasa está capitalizada diariamente.
- La tasa de interés y el tiempo siempre deben estar expresados en la misma unidad de medida (siempre manda el periodo de capitalización).

### Fórmulas

```
Valor futuro

S = C × (1 + (TN / m))^n

TN: tasa nominal contratada
m: capitalización
n: número de periodos de tiempo según la capitalización
```

```
Valor presente

C = S / (1 + (TN / m))^n

TN: tasa nominal contratada
m: capitalización
n: número de periodos de tiempo según la capitalización 
```

```
Tasa nominal contratada

TN = m × [(S / C)^(1/n) - 1]

m: capitalización
S: valor futuro
C: capital
n: número de periodos de tiempo según la capitalización
```

```
Periodos de tiempo

n = [ln(S / C)] / [ln(1 + (TN / m))] periodos de tiempo

ln: logaritmo natural
S: valor futuro
C: capital
TN: tasa nominal contratada
m: número de periodos de tiempo
```
